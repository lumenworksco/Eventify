# Project template front-end

## Dotenv

To get this demo up and running, you'll need to create a **.env** file in you root project directory (on the same level as .gitignore).
The contents should look like this:

```properties
NEXT_PUBLIC_API_URL = http://localhost:3000
```

This will be the API URL where your back-end is running.

## Install and start Next.js app

Follow the instructions given in class to install Next.js.
